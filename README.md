# ML-SupervisedLearning

## Linear Regression From Scratch

Проект посвящён реализации и исследованию линейных моделей машинного обучения с использованием Python и NumPy без готовых реализаций алгоритмов.

В рамках проекта были реализованы:

* линейная регрессия,
* Ridge, Lasso и ElasticNet-регрессия,
* аналитическое решение,
* градиентный и стохастический градиентный спуск,
* методы нормализации признаков,
* генерация полиномиальных признаков,
* оценка качества моделей и анализ переобучения.

Также проводилось сравнение собственных реализаций моделей с аналогами из библиотеки Scikit-learn.

---

## Возможности проекта

### Реализованные модели

В проекте реализованы основные линейные модели для решения задач регрессии:

* Linear Regression,
* Ridge Regression,
* Lasso Regression,
* ElasticNet Regression.

Для обучения моделей использовались как аналитические методы решения, так и оптимизационные алгоритмы:

* Gradient Descent,
* Stochastic Gradient Descent (SGD).

---

### Предобработка данных

В рамках проекта были реализованы и протестированы основные этапы подготовки данных:

* обработка пропущенных значений,
* генерация дополнительных признаков,
* нормализация и стандартизация данных,
* создание полиномиальных признаков,
* подготовка данных для обучения и тестирования моделей.

Также отдельно исследовалось влияние масштабирования признаков на качество моделей и скорость сходимости алгоритмов.

---

### Метрики качества

Для оценки качества моделей использовались основные метрики регрессии:

* MAE,
* RMSE,
* R²,
* MAPE.

Проводилось сравнение качества моделей на обучающей и тестовой выборках, а также анализ переобучения и устойчивости моделей.
